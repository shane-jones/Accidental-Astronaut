# Accidental-Astronaut
A visual novel created in Ren'Py as a student assignment.





# Installation
**Instructions for installation:** 
1. Unzip the file in the desired directory 
2. Open the AccidentalAstonaut-1.0-all folder
3. Double click the AccidntalAstonaut.exe
4. Start playing or set preferences





# File structure 
**Folder structure within the game folder** 
**Declarations:** 
Contains a declarations file for character and audio definitions, and image statements. 
**Scripts:** 
Contains scripts for initialisation and debugging as well as one script per story scene. 
**Images:** 
Contains sub-folders for backgrounds, characters and objects. 
**Sounds:** 
Contains sub-folders for effects and music.
